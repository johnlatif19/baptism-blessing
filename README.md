# Baptism Blessing

A luxurious Coptic Orthodox Holy Baptism invitation website featuring event details, countdown, gallery, videos, Google Maps, and a secure family dashboard.

## Features

- 🎨 **Premium Design**: Elegant, luxurious, warm, and joyful design with gold accents
- 🌓 **Dark/Light Mode**: Full dark and light theme support across all pages
- 🌐 **Bilingual**: Arabic and English language support with RTL/LTR support
- 📱 **Responsive**: Perfect on desktop, tablet, and mobile devices
- 🔐 **Secure Authentication**: JWT-based authentication with protected dashboard
- 📸 **Gallery Management**: Upload and manage images with Cloudinary integration
- 🎥 **Video Management**: Add and manage embedded YouTube videos
- ⏰ **Event Timeline**: Manage event timeline items
- 📍 **Google Maps**: Embedded map with location details
- 📊 **Dashboard**: Full admin dashboard to manage all content
- 🔗 **QR Code**: Generate QR code for easy invitation sharing
- 🕊️ **Countdown**: Live countdown to the event

## Tech Stack

### Frontend
- HTML5
- CSS3 (with CSS Variables for theming)
- Vanilla JavaScript
- Lucide Icons
- Google Fonts (Playfair Display, Cormorant Garamond, Poppins)

### Backend
- Node.js
- Express.js
- JWT Authentication
- bcryptjs for password hashing

### Database
- Firebase Firestore

### Storage
- Cloudinary

### Deployment
- Vercel
